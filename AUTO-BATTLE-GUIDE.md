# 自动战斗系统使用指南

## 🎯 系统概述

土豆地城现在支持**全自动战斗系统**！玩家无需手动指挥每回合战斗，系统会根据你选择的战斗风格自动进行战斗。

---

## ⚔️ 战斗风格

### 1. 激进型 (Aggressive)
- **特点**：高攻击、低防御
- **适用场景**：快速刷怪、自信碾压
- **属性加成**：
  - 攻击加成：+2
  - 防御加成：-1
  - 技能使用率：70%
  - 治疗优先级：低
  - 逃跑阈值：10% HP

**适合**：高等级角色碾压低级地城

### 2. 平衡型 (Balanced)
- **特点**：攻守均衡
- **适用场景**：日常探索、稳定发育
- **属性加成**：
  - 攻击加成：+0
  - 防御加成：+0
  - 技能使用率：50%
  - 治疗优先级：中
  - 逃跑阈值：20% HP

**适合**：大部分情况，推荐新手使用

### 3. 保守型 (Defensive)
- **特点**：高防御、会治疗
- **适用场景**：挑战高难度、生存优先
- **属性加成**：
  - 攻击加成：-1
  - 防御加成：+2
  - 技能使用率：30%
  - 治疗优先级：高（HP < 40%时优先治疗）
  - 逃跑阈值：30% HP

**适合**：挑战高等级地城、血量危险时

## 🎮 使用方法

### 1. 设置战斗风格

访问战斗设置页面：
```
http://localhost:10001/battle-settings.html?characterId=YOUR_CHARACTER_ID
```

选择你想要的战斗风格，点击"保存设置"。

### 2. 开始地城探索

在探索地城时，系统会自动：
1. 根据你的战斗风格决定战斗策略
2. 自动计算每回合的行动
3. 处理攻击、技能、治疗、逃跑等决策
4. 生成完整的战斗日志

### 3. 查看战斗结果

战斗结束后，你可以看到：
- ✅ 完整的战斗日志（每回合详细记录）
- ✅ 战斗统计（回合数、剩余HP/MP）
- ✅ 获得的奖励（经验、金币、装备掉落）
- ✅ 是否升级或获得装备

## 📊 战斗机制详解

### 属性影响

- **力量 (STR)**：影响物理攻击伤害
- **敏捷 (DEX)**：影响逃跑成功率
- **体质 (CON)**：影响防御力
- **智力 (INT)**：影响魔法技能伤害
- **感知 (WIS)**：影响暴击率
- **魅力 (CHA)**：影响装备掉落率

### 战斗流程

1. **遭遇怪物** → 系统自动进入战斗
2. **自动战斗** → 根据战斗风格AI自动决策
3. **战斗结束** → 显示结果和奖励
4. **继续探索** → 自动进入下一个房间

### 特殊机制

- **暴击系统**：掷出20自然暴击，感知≥15时暴击率+5%
- **逃跑机制**：HP低于阈值时有几率逃跑，敏捷影响成功率
- **技能系统**：自动使用职业技能（需要MP）
- **治疗优先**：保守风格会在HP低时优先治疗
- **装备掉落**：基础30% + 地城等级加成 + 魅力加成

## 🔄 API 接口

### 探索房间（自动战斗）

```http
POST /api/dungeon/explore
Content-Type: application/json

{
  "characterId": 1,
  "runId": 123,
  "battleStyle": "balanced"  // 可选，不传则使用角色保存的设置
}
```

**响应示例**：
```json
{
  "success": true,
  "type": "combat_victory",
  "description": "⚔️ 遭受 Slime！战斗胜利！\n获得 8 经验，5 金币",
  "battleLog": ["=== 第 1 回合 ===", "⚔️ 玩家攻击...造成 5 点伤害", "👹 Slime 攻击！造成 2 点伤害"],
  "reward": {
    "exp": 8,
    "gold": 5,
    "isCrit": false
  },
  "levelUp": false
}
```

### 获取战斗设置

```http
GET /api/character/:characterId/battle-settings
```

### 保存战斗设置

```http
POST /api/character/:characterId/battle-settings
Content-Type: application/json

{
  "battleStyle": "aggressive"  // "aggressive" | "balanced" | "defensive"
}
```

## 💡 使用建议

### 推荐策略

1. **新手期（等级1-3）**
   - 使用平衡型
   - 挑战推荐等级的地城
   - 积累装备和金币

2. **成长期（等级4-7）**
   - 根据地城难度切换风格
   - 低级地城用激进速刷
   - 高级地城用保守稳打

3. **挑战期（等级8+）**
   - 高难度地城用保守型
   - Boss战前恢复HP/MP
   - 合理利用装备加成

### 效率优化

- **刷经验**：用激进型快速清理低级地城
- **刷装备**：平衡型稳定探索，提高魅力增加掉落
- **挑战Boss**：保守型确保存活，必要时带足药水

## 🐛 常见问题

**Q: 战斗失败会丢失装备吗？**
A: 不会。战斗失败只会结束当前地城探索，不会丢失已获得的装备和金币。

**Q: 可以在战斗中途改变策略吗？**
A: 目前不行。战斗是自动进行的，建议在进入地城前设置好合适的战斗风格。

**Q: 逃跑成功后还能继续探索吗？**
A: 可以。逃跑成功后会跳过当前怪物，你可以继续探索下一个房间。

**Q: 战斗日志会保存吗？**
A: 战斗日志只在当前会话显示，建议截图保存精彩战斗！
