# ⚔️ 土豆地城 - 文字战斗系统设计

## 一、战斗界面设计

战斗采用**文字冒险日志**方式，让玩家能看到完整的战斗过程：

```
回合 1：你遭遇【史莱姆】
HP: 8/8  | 防御力：6  | 闪避率：10%
> 史莱姆发起了攻击！
> 你选择攻击动作...
> 你用【巨剑】攻击！
> 掷中！命中史莱姆(6-10-4)。
> 造成 4 点伤害（力量+1）
> 史莱姆受到 2 点伤害。
> 史莱姆发动【粘液】！
> 史莱姆的【粘液】效果让你受到 2 点伤害，持续 1 回合（命中时50%减速）
> 史莱姆还剩 6 HP。

回合 2：史莱姆行动
> 史莱姆发动【撕咬】！
> 你用【普通】攻击！
> 技能检定：敏捷(DEX+1) vs 洞穴环境 +5 = 成功！
> 造成 8 点伤害（力量+2）
> 史莱姆发动【粘液】！
> 史莱姆受到 2 点伤害。

回合 3：战斗结束
> 【史莱姆】被击败！
> 获得：5 金币、10 经验
> 史莱姆死后掉落了 1 个【小型宝箱】，里面装着【粗戒指】

=======================================
当前状态
角色：土豆骑士
HP: 8/8
金币：15
背包：【粗戒指】(+1 魅力)
```

## 二、战斗公式

### 1. 命中判定
```
命中率 = 50 + D20 + (玩家攻击修正 - 敌人防御) - (怪物敏捷 + 地城环境修正)

命中率 = Math.min(95, 50 + 攻击骰 + (力量修正 - 怪物防御) - (怪物敏捷 + 地城环境修正))
```

### 2. 伤害计算
```
基础伤害 = 玩家攻击力 - 怪物防御
暴击倍率 = 1.5（命中时暴击）

玩家伤害 = 基础伤害 × 暴击倍率
暴击伤害 = 玩家攻击力 × 2（暴击时）

最终伤害 = 玩家防御 - 玩家伤害
怪物受到伤害 = Math.max(0, 玩家伤害 - 怪物防御)

怪物受到暴击伤害 = 怪物基础伤害 × 2
```

### 3. 闪避检定
```
闪避率 = 50 + D20 + (玩家敏捷 - 怪物敏捷) - (玩家防御) - (怪物敏捷 + 地城环境修正)

闪避成功 = 闪避骰 > 闪避率
```

### 4. 技能效果
```
【粘液】(持续 1 回合伤害) - 史莱姆的粘液，命中后让玩家下回合受到 2 点伤害
【冰霜】(爆发伤害) - 3 回合内每回合造成 50% 额外伤害
【撕咬】(连续攻击) - 每次命中后增加 10% 暴击率，最多 50%
【燃烧】(火焰) - 3 回合内造成 +50% 伤害
【分裂】(复制) - 普通攻击时，如果怪物 HP 少于 50%，有 20% 几率分裂出小史莱姆
```

## 六、界面元素

- 战斗日志区域：实时文字显示战斗过程
- 行动按钮区域：普通攻击、重击、技能、道具
- 状态栏：HP、金币、经验条
- 怪物信息面板：显示怪物 HP、AC、类型
- 战斗结果面板：战斗结束、失败、胜利信息

- 图标：使用 ASCII 艺术增强视觉效果
