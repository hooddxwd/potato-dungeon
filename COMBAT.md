# ⚔️ 土豆地城 - 战斗系统设计

## 一、核心机制

### 1.1 属性系统

| 属性 | 作用 |
|------|------|
| HP | 生命值，0 时战斗失败 |
| 力量 STR | 物理伤害 = (力量 - 敌人防御/2) + 骰子 |
| 敏捷 DEX | 闪避率 = (敏捷 - 敌人敏捷) / 2 + 10% |
| 体质 CON | HP上限 = 10 + (体质-10) × 2 |
| 智力 INT | 魔法伤害 / 技能成功率 |
| 感知 WIS | 陷阱发现率 / 暴击率 |
| 魅力 CHA | 掉落率 / NPC对话 |

### 1.2 伤害公式

```
物理伤害 = max(1, 攻击方力量 - 防御方防御/2 + D6骰子)
魔法伤害 = max(1, 攻击方智力 - 防御方意志/2 + D8骰子)
暴击伤害 = 伤害 × 2 (感知 ≥ 15 时 +5% 暴击率)
```

## 二、战斗流程

### 2.1 回合制

```
每回合可用动作：
1. ⚔️ 攻击 - 普通物理攻击
2. 🛡️ 防御 - 本回合受到伤害 -50%
3. 🎯 技能 - 使用职业技能（需要MP）
4. 🧪 道具 - 使用背包物品
5. 🏃 逃跑 - 30% 成功率（敏捷越高越高）
6. 📖 观察 - 查看敌人属性
```

### 2.2 怪物AI

| 类型 | 行为 |
|------|------|
| 激进型 | 优先攻击，有机会连击 |
| 防守型 | 优先防御，等待反击 |
| 均衡型 | 根据血量选择策略 |
| 狡猾型 | 30% 概率欺骗 / 逃跑 |

## 三、地城结构

### 3.1 每层结构

```
地城 X 层
├── 房间 1（入口）→ 房间 2 → 房间 3 → ... → 房间 N（Boss）
│   ├── 房间类型：战斗/宝箱/事件/陷阱/休息/商店
│   ├── 每个房间 1-3 个怪物
│   ├── 可能有隐藏房间（感知检定）
│   └── 探索度系统
```

### 3.2 房间类型

| 房间 | 描述 | 内容 |
|------|------|------|
| 战斗房 | 遭遇怪物 | 1-3 个怪物 |
| 宝箱房 | 宝藏 | 金币/装备/道具 |
| 事件房 | 随机事件 | 商人/谜题/祭坛 |
| 陷阱房 | 危险 | 各种陷阱 |
| 休息房 | 安全区 | 恢复 HP |
| Boss房 | 最终挑战 | Boss 怪物 |

## 四、怪物设计

### 4.1 怪物属性

```javascript
{
  name: "史莱姆",
  hp: 10,           // 生命值
  atk: 5,           // 攻击力
  def: 2,            // 防御力
  speed: 8,           // 速度（先手判定）
  exp: 10,           // 击杀获得经验
  gold: 5,           // 击杀获得金币
  loot: 0.3,         // 掉落率
  type: "beast",     // 类型（影响抗性）
  skill: "分裂",     // 特殊技能
  ai: "aggressive"   // AI类型
}
```

### 4.2 技能系统

| 技能类型 | 效果 |
|----------|------|
| 连击 | 50% 概率再次攻击 |
| 吸血 | 造成伤害的 30% 恢复自身 |
| 毒液 | 每回合 -2 HP，持续 3 回合 |
| 火焰吐息 | 范围伤害，无法防御 |
| 魅惑 | 20% 概率让敌人无法行动 1 回合 |
| 重生 | 死亡时恢复 50% HP（限1次） |

## 七、地城进度

### 7.1 探索度

- 每个房间探索 = +10% 探索度
- 100% 探索度 = 开启隐藏房间
- 完整通关 = 获得通关奖励

### 7.2 楼层传送

- 通关当前层 → 获得传送门
- 可以直接传送回上一层
- 每层只能使用 1 次
