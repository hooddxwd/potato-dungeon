# 🎮 土豆地城 - 完整游戏策划报告

---

## 一、项目概述

### 1.1 核心概念
- **Roguelike 地城探险游戏**：随机生成地城，永久死亡
- **游戏化 Todo 系统**：完成任务获得属性经验，升级属性
- **文字战斗系统**：类似经典文字 RPG，详细描述战斗过程
- **多职业系统**：战士、法师、盗贼等
- **7 个地城**：从简单到深渊

### 1.2 目标用户
- 喜欢回合制游戏的玩家（25-40岁）
- 休闲玩家（每天 15-30 分钟）
- 追求高难度挑战

---

## 二、游戏循环

### 2.1 核心循环
```
玩家登录 → 创建角色 → 探索地城 → 战斗/冒险 → 升级 → 挑战更高地城
```

### 2.2 资源循环
```
完成 Todo → 获得属性经验 → 升级/解锁新地城 → 获得金币/装备 → 购买消耗品
```

## 四、战斗系统设计

### 4.1 属性系统 (DND 6大属性)
```
| 属性 | 作用 | 升级影响 |
|------|------|
| 力量 STR | 物理攻击、近战伤害 | +1 体力 |
| 敏捷 DEX | 闪避率、远程攻击 | +1 移动力 |
| 体质 CON | HP 上限、防御加成 | +2 HP/级 |
| 智力 INT | 魔法攻击 | +1 技能使用 |
| 感知 WIS | 陷阱发现率、Boss 检定 | +1 探索奖励 |
| 魅力 CHA | 掉落率加成、NPC 好感度 | +1 任务奖励 |
```

### 4.2 战斗公式

**命中公式**
```
命中率 = 50 + D20 + (玩家攻击修正 - 敌人防御)
        = 50 + D20 + (STR - 10)/2 + (DEX - 10)/2 - (怪物理防御/2)
        
暴击率 = 5 + (感知 >= 15 ? +5% : 0)
暴击伤害 = 普通伤害 × 2
```

**伤害公式**
```
基础伤害 = 玩家攻击力 - 怪物防御
          = MAX(1, 力量 - 怪物防御)
          = MAX(1, 力量 - 防御力)
          
暴击伤害 = (命中时暴击 ? 基础伤害 × 2 : 基础伤害)
        
最终伤害 = MAX(0, 基础伤害 - 怪物防御)
          = MAX(0, 力量 - 防御力 - 防御力)
```

### 4.3 状态效果

| 效果 | 持续时间 | 机制 |
|------|------|------|
| 粘液 | 1 回合 | 玩家无法行动、命中-50% |
| 燃烧 | 2 回合 | 魔法 +50% 伤害 |
| 恐惧 | 3 回合 | 命中 -50% 暴击 |
| 中毒 | 3 回合 | 持续伤害、命中-30% |
| 混乱 | 持续 1 回合 | 无法使用技能、随机行动 |

## 五、装备系统

### 5.1 装备品质
```
品质 | 稀有度 | 掉落率 | 属性加成 |
|------|------|--------|----------|
| 普通 | 30% | +1-2 | 灰色 |
| 优秀 | 15% | +3-4 | 绿色 |
| 稀有 | 10% | +5-7 | 蓝色 |
| 史诗 | 5% | +8-12 | 紫色 |
| 传说 | 1% | +15-20 | 橙色 |
```

### 5.2 装备类型
```
类型 | 可装备数量 | 属性加成 |
|------|------|--------|
| 武器 | 1 | STR +1-3 |
| 盔牌 | 1 | CON +1-3 |
| 戒指 | 1 | WIS +1-2 |
| 饰品 | 1 | CHA +1 |
```

### 5.3 装备掉落
```
掉落条件：
- 普通怪物：30% 几率
- Boss 战：100% 几率
- Boss 越杀：必定掉落装备

掉落公式：
- 地城等级影响品质上限（地城 1 只能掉普通，地城 8 可掉传说）
- 稀有度影响装备属性加成
```

## 六、职业系统 (V2.0 - 未来扩展)

### 6.1 职业列表

| 职业 | 核心属性 | 技能 | 特殊效果 |
|------|------|--------|----------|
| 战士 | STR++, CON++ | 冲锋、盾牌格挡 |
| 法师 | INT++, WIS++ | 火焰术、暴击加成 |
| 盗贼 | DEX++, STR++ | 潜行、暴击、开锁 |
| 牧师 | WIS++, CHA++ | 治愈术、状态解除 |

### 6.2 职业系统
```
每级获得技能点数：3 点
技能树：被动技能 + 1 个主动技能

技能示例：
- 战士：【冲锋】（消耗 10 MP，下次攻击 +20%）
        【盾牌格挡】（消耗 5 MP，下次受到伤害 -30%）
- 法师：【火球术】（消耗 15 MP，范围 +3 火焰伤害）
        【冰冻术】（消耗 20 MP，攻击 +10%，敌人速度 -50%，持续 1 回合）
- 盗贼：【背刺】（消耗 5 MP，暴击率 +20%）
        【潜行】（消耗 10 MP，命中 +30%，敌人无法发现）
```

**总结**

土豆地城是一个轻量级、易上手、深度的 Roguelike 文字冒险游戏，融合了 DND 规则、现代游戏设计理念（任务系统）、文字战斗和社交元素。

当前已完成功能可以作为 MVP（最小可行产品）提供给玩家体验，后续可根据用户反馈和运营数据持续优化和扩展新功能。
